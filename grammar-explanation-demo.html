📚 文法説明コンポーネント

grammar.jsonのデータを美しく表示するためのコンポーネント

✨ 主な機能

  • grammar.jsonのデータを自動読み込み・表示
  • 前後の文法項目へのナビゲーション
  • 文法項目のインデックス表示と直接ジャンプ
  • 印刷機能（印刷用CSS対応）
  • テキストコピー機能
  • 全画面表示モード
  • キーボードナビゲーション（←→キー）
  • レスポンシブデザイン対応
  • ダークモード対応
  • HTMLタグ付きテキストの適切なレンダリング

📖 使用方法

このコンポーネントを使用するには、以下のHTMLコードをページに追加してください：

<!-- 文法説明コンポーネントを読み込み -->
<div id="grammar-explanation-container"></div>

<script>
// コンポーネントを動的に読み込み
fetch('/components/grammar-explanation.html')
  .then(response => response.text())
  .then(html => {
    document.getElementById('grammar-explanation-container').innerHTML = html;
  });
</script>

または、既存のページに直接コンポーネントを埋め込む場合：

<!-- 既存のコンテンツ -->
<div class="your-content">
  <!-- ここに文法説明コンポーネントを配置 -->
  <div id="grammar-explanation-container"></div>
</div>
